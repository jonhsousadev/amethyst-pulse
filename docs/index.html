Jonh Sousa • Otimizado para Astigmatismo • v1.0.4

Por que Amethyst?

O efeito de "halo" em telas escuras é um desafio real para desenvolvedores com astigmatismo. Este tema utiliza tons profundos de ametista para suavizar a transição entre o texto e o fundo, eliminando o contraste agressivo do preto absoluto.
