# 🟣 Amethyst Pulse

**Amethyst Pulse** é um tema premium para Visual Studio Code, desenhado para oferecer uma experiência de programação profunda, tecnológica e, acima de tudo, confortável.

Inspirado na clareza das pedras preciosas e no conceito de *High-Contrast Softness*, este tema substitui os contrastes agressivos por uma paleta equilibrada de **Roxo Ametista**, **Laranja Âmbar** e **Creme pulse**.

---

## ✨ Destaques

* 👁️ **Otimizado para Astigmatismo:** Fundo roxo profundo (não preto puro) e texto em tom creme para reduzir o efeito de "halo" e a fadiga ocular.
* ⚡ **Hierarquia Lógica:** Palavras reservadas (`if`, `else`, `return`, `for`, `while`) em laranja vibrante para identificação instantânea do fluxo.
* 💎 **Estética Amethyst:** Interface consistente, desde as badges vibrantes até às guias de indentação sutis.
* 🎨 **Foco em Web Development:** Sintaxe refinada para HTML, CSS, JavaScript e TypeScript.

## 🚀 Instalação

### Opção 1: Instalação Manual (Recomendado para Devs)

1. Faça o download ou clone este repositório.
2. Localize a sua pasta de extensões do VS Code:
   * **Windows:** `%USERPROFILE%\.vscode\extensions`
   * **macOS / Linux:** `~/.vscode/extensions`
3. Copie a pasta `amethyst-pulse` para dentro desse diretório.
4. Reinicie o VS Code.
5. Pressione `Ctrl + K` + `Ctrl + T` e selecione **Amethyst Pulse**.
