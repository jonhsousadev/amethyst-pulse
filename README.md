## ЁЯЪА Installation

### Option 1: Manual Installation (Recommended for Devs)

1. Download or clone this repository.
2. Locate your VS Code extensions folder:
   * **Windows:** `%USERPROFILE%\.vscode\extensions`
   * **macOS / Linux:** `~/.vscode/extensions`
3. Copy the `amethyst-pulse` folder into this directory.
4. Restart VS Code.
5. Press `Ctrl + K` + `Ctrl + T` and select **Amethyst Pulse**.

### Option 2: Via Marketplace

[Install from Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=JonhSousa.amethyst-pulse&ssr=false#overview)

## ЁЯЯг Zed Installation

### Option 1: Via Zed Extensions (Recommended)

1. Open Zed and press `Ctrl + Shift + X` (or `Cmd + Shift + X` on macOS) to open the Extensions panel.
2. Search for **Amethyst Pulse**.
3. Click **Install**.
4. Open the theme picker with `Ctrl + K` + `Ctrl + T` (or `Cmd + K` + `Cmd + T`) and select **Amethyst Pulse**.

### Option 2: Manual Installation

1. Clone the Zed extension repository:
   ```bash
   git clone https://github.com/jonhsousadev/amethyst-pulse
   ```
2. Copy the folder called "zed" from the cloned folder into your Zed extensions directory:
   * **macOS / Linux:** `~/.local/share/zed/extensions/installed/amethyst-pulse`
   * **Windows:** `%APPDATA%\Zed\extensions\installed\amethyst-pulse`
3. Restart Zed.
4. Open the theme picker with `Ctrl + K` + `Ctrl + T` (or `Cmd + K` + `Cmd + T`) and select **Amethyst Pulse**.
